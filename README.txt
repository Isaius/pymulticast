Instruções para execução do trabalho Protocolo Multicast: Bounty Hunter

1. Certifique-se de ter instalado alguma versão do Python 3.x nas máquinas
2. Nas máquinas servidoras:
    2.1. Execute em cada uma delas o programa "Receiver.py" sem argumentos:
            $ python3 Receiver.py
3. Na máquina cliente:
    3.1. Execute o programa "Sender.py" sem argumentos:
            $ python3 Sender.py
    3.2. Digite no prompt de comando expressões algébricas simples como:
            2+2
            3+4*5
            10/2-3
    3.3. Veja o resultado ser enviado em alguns instantes
